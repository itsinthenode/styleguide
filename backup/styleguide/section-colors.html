3.3 #Colors.Functions color($name)

  • $name
    Name of the color.

Sass function that returns specified color hex value from color palette.

Usage:

.my-component {
  background-color: color("green");
  color: color("metal");
}

3.4 #Colors.Mixin color($bgColor, $fontColor)

  • $bgColor
    Name of background color.
  • $fontColor
    Name of font color.

Sass mixin that sets background color and font color.

Usage:

.my-component {
  @include color("carbon", "white");
}
