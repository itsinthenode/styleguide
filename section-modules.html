4 #Modules Modules

logical components that can consist of multiple tags

4.1 #Modules.Flash Messages Flash messages

for message popups

Example
This is a flash message
  <div class="flash">This is a flash message</div>

4.2 #Modules.Phone Wrapper Phone wrapper

displays a phone mockup.

Example 
  <div class="phone-wrapper"></div>
  <!-- also available as SASS mixin -->

4.3 #Modules.Profile Image Profile Image

for square images

Examples
Default styling
.profile-image--small
small version
.profile-image--tiny
tiny version 
  <img class="profile-image [modifier class]" src="http://lorempixel.com/300/300/people/1">
